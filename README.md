# SREgent
本项目旨在构建一个ebpf赋能的运维智能体，旨在探索运维的无限可能。它能够与文件系统无缝衔接，提供基本Linux系统操作代理能力；它能够作为运维专家，回答专业运维问题；它能够辅助用户进行系统观测，并调用分析工具进行运维故障分析，给出修复建议。ebpf作为新一代的可观测性工具，给智能体提供系统深度观测能力，并通过一系列基于ebpf的基础设施，丰富智能体的运维工具箱。

## SRE的独特优势
1. 它是第一个提出融合ebpf可观测性进行智能体设计的项目
2. 它构建了一套完整的运维智能体测评框架SREgent-test
3. 它通过创新智能体的架构设计，工具设计与RAG设计，最大程度地在运维可靠性的要求下，释放智能潜力。
4. 在实验的NFS环境中，它能够做到与系统高度集成，代理用户进行文件系统运维管理


> Extra Bonus: 它展示了一个完整的智能体开发流程所需的测试实验与架构探索
